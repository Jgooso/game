Catch Up

by: Josh Goodman

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I found it online and it seemed easy and fun to make.

2. What I Learned

I learned to make the AI harder because it was to easy.

3. Architecture of My Game

When the button is clicked the actionListener is in the panel and calls methods from the game manager to react to the buttons being clicked then responds to the game manager

4. Technical Implementation

The computer AI first checks if any button would put its score above or equal to the definate winning score. After that it picks the highest number that would keep its score under the players score. Then it either picks a number randomly or just picks the largest remaining number

5. Most Challenging Part

The most challenging part was transferring the data between the gameAI and manager because I couldn't just return all the data. I needed to add set and get methods to pass data between the different classes.

6. Future Fixes and Updates

I would add the ability to change the amount of buttons on the screen from any amount 2-12.
